# League Build Tool

A .NET Core application that helps players optimize their League of Legends item builds.

## Features

- Fetch real-time item data from Riot Games API
- Champion build management
- Item recommendations

## Project Structure

- **LeagueBuildTool.App**: Main application entry point
- **LeagueBuildTool.Core**: Core business logic and models
- **LeagueBuildTool.Tests**: Unit tests

## Requirements

- .NET 8.0 SDK
- Visual Studio 2022 or VS Code

## Getting Started

1. Clone the repository
2. Open `LeagueBuildTool.sln` in Visual Studio
3. Build and run the solution

## Running Tests

```bash
dotnet test
```
